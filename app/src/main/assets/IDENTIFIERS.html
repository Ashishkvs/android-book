Identifiers

Any name in java is termed as Identifier.It can be the name of a variable,method or class.

They are various rules to define an Identifier in Java:

  • Rule1: The only alowed charecters in java identifiers are:- 
    • a to z
    • A to Z
    • 0 to 9
    • _(underscore)
    • $
  • Rule2: An Identifier should not start with any digits.
  • Rule3:Identifiers are case sensitive.
  • Rule4:There is no length limit for the identifiers, but it is not recommendable to create an identifiers with length more than 15.
  • Rule5:We cannot use reserved words as Identifiers. 

* Note:Reserve Words :In java some identifiers are reserved for specific functionality or meaning.
